Introduction to testdat

Danny Smith

2018-02-13

testdat is an R package extending the unit testing framework of testthat to data frames.

Getting started

As an extension of testthat, testdat uses the same basic testing framework. For a simple introduction to testthat, see the section on testing in R packages.

The main addition in testdat is a set of expectations designed for testing data frames and a mechanism to attach a data frame to a context.

The different classes of data expectation are:

  • [generic-expectations]
  • [conditional-expectations]
  • [value-expectations]
  • [datacomp-expectations]
